# 衣启盲盒 · 服装盲盒订阅原型

纯静态 HTML 移动端交互原型（单文件，无任何依赖），演示服装盲盒订阅完整流程：

1. **盲盒列表**：展示各品类盲盒（T恤 / 衬衫 / 西裤 / 卫衣 / 连衣裙），含商品样例图、名称、价格、运费。样例图仅表达风格，实际发货为同品类惊喜款。
2. **盲盒详情**：浏览更多样例图，点选最多 9 张作为发货喜好依据，可随时更换。
3. **订阅设置**：选择尺码、填写收货地址、选择发货频率与订阅期数，确认订阅。
4. **我的订阅**：查看订阅进展（发货进度、下期发货时间），随时调整喜好样图。

订阅数据保存在浏览器 localStorage 中，刷新不丢失。

## 本地预览

直接用浏览器打开 `index.html`，或：

```bash
python3 -m http.server 8000
# 访问 http://localhost:8000
```
